# INCOFY
FREE AND OPEN SOURCE SVG ICONS FOR FRAMEWORKS AND LANGUAGES

- [x] Collections of svg icons : frameworks - languages - tools
- [x] Convert svg to png/jpg (using canvas)
- [x] Svg editor

# Notes

  - icons-frameworks
  - icons-languages
  - icons-tools
- All pull requests are welcome, feel free.
